# 🚀 Instalación

1. Clonar el repositorio.

```bash
git clone https://github.com/Xtobal-Monroy-Paredes/SistemaHospitalJava.git
```

2. Abrir el proyecto con NetBeans.

3. Ejecutar el proyecto.

La aplicación utiliza una base de datos **MySQL alojada en Railway**, por lo que no es necesario instalar ni configurar una base de datos local mientras la conexión a Railway esté disponible.

---

# 🗄 Base de datos

La base de datos se encuentra alojada en **Railway**.

También se incluye el archivo **BDHospital.sql** como respaldo para recrear la base de datos en un servidor MySQL local si fuera necesario.

# 🧩 Arquitectura

El proyecto implementa:

- Programación Orientada a Objetos (POO)
- Patrón DAO
- JDBC para la conexión a la base de datos
- MySQL como gestor de base de datos
- Java Swing para la interfaz gráfica

---

# 🎯 Objetivo

Desarrollar un sistema hospitalario que permita administrar pacientes, doctores y citas médicas mediante una interfaz gráfica conectada a una base de datos MySQL.
